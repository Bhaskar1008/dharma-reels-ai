# DharmaReels AI 🎬

DharmaReels AI is an AI-powered video generation platform that converts Telugu mythology scripts into engaging short videos automatically.

## 🚀 Features
- 🧾 Script-to-Video generation (Telugu supported)
- 🎙️ AI voice synthesis (TTS)
- 🖼️ AI image generation for scenes
- 🎬 Automated video rendering using FFmpeg
- 📜 Subtitle generation (SRT)
- 🎵 Background music integration

## 🏗️ Tech Stack
- Frontend: React (Vite) + Tailwind — see `frontend/README.md`
- Backend: Node.js (Express)
- Video Engine: FFmpeg
- AI Services: TTS + Image Generation APIs

## Run locally

1. **Backend** (from `backend/`): `npm install` → `npm run dev` (default `http://localhost:3001`).
2. **Frontend** (from `frontend/`): `npm install` → copy `frontend/.env.example` to `frontend/.env` → `npm run dev` (default `http://localhost:5173`).

## 🎯 Use Case
Create YouTube Shorts, Instagram Reels, and TikTok videos from mythology stories like Mahabharata, Ramayana, and more.

## 🔥 Vision
To simplify content creation by enabling users to generate high-quality AI videos with minimal effort.
